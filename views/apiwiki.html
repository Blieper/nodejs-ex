Introduction

Welcome to our API! Everything here is based on HTTP requests. Every API request is based on the link: 

http://nodejs-mongo-persistent-gmodcarregistration.193b.starter-ca-central-1.openshiftapps.com/api?

To use the API, you must use your own unique API key. You can find that in your account info. API parameters do not need a certain order. All parameters are seperated by the '&' symbol.

Here's an example of the API that returns a random car license, based on a region named 'Spitsburg': 

http://nodejs-mongo-persistent-gmodcarregistration.193b.starter-ca-central-1.openshiftapps.com/api?token=XXXXXXXXXXXXXX&randlicense&host=Spitsburg

As you can see, there are 3 parameters:

  1. token=XXXXXXXXXXXXXX
  2. randlicense
  3. host=Spitsburg

Not every parameter needs a value, like the 'host' parameter. If you wish to send an array for a certain parameter you can do this: 

a=1&a=2&a=3

This would make it so the API recieves a as array {1,2,3}
